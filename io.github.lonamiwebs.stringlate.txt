Summary:Help translating FOSS applications
Description:
Fetches remote GitHub repositories containing the source code for Android
applications downloading the available `strings.xml` files in them for you
to translate the strings to any other locale, in order to help the authors
localize their applications into other languages.

For more information and examples please see the README.md file on the website!
